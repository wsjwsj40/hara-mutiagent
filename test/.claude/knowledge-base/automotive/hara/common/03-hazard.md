# HARA 整车级危害产生
本文件用于指导 HARA 分析中的**整车级危害产生**。根据上一阶段分析生成的故障描述，展开除不适用以外的所有功能故障，映射到对应的整车级危害上。

## 1. 映射原则
整车级危害必须来自预定义的整车级危害库 `vehicle_hazards.json`，不能自由编写。

## 2. 危害选项定义

在使用 `vehicle_hazards.json` 时，注意以下容易混淆的选项区分：

### 核心判断维度
判断整车危害时，从以下两个维度分析：
1. **车辆运动状态**：车辆是"自己动了"还是"想动动不了"？
2. **驾驶员意图**：驾驶员是否主动请求运动？

### 纵向运动详细定义
- **非预期的纵向移动**：车辆在驾驶员无运动请求时发生移动
  - 核心特征：车辆"自己动了"（溜车、滑动、非预期的位移）
  - 适用场景：驻车失效、保持失效、约束失效、锁止失效
  - 判断方法：功能失效后，车辆是否会因为重力、惯性或外力而产生非预期位移？

- **无法纵向移动**：车辆在驾驶员请求运动时无法实现
  - 核心特征：车辆"想动动不了"
  - 适用场景：动力系统失效、驱动系统失效、扭矩传递中断
  - 判断方法：功能失效后，驾驶员主动请求前进/后退但车辆无响应？

- **非预期的加速**：车辆获得非请求的向前加速力
  - 核心特征：车辆"意外加速"
  - 适用场景：非预期动力输出、扭矩意外输出

- **非预期的失去加速**：车辆失去加速能力
  - 核心特征：请求加速但加速能力不足或丧失
  - 适用场景：动力系统失效、扭矩输出不足

- **非预期的减速**：车辆获得非请求的减速力
  - 核心特征：车辆"意外减速"
  - 适用场景：非预期制动、行驶中EPB意外拉起

- **非预期的失去减速**：车辆失去减速能力
  - 核心特征：请求减速但减速能力不足或丧失
  - 适用场景：制动系统失效、制动力不足

- **非预期车辆减速上的降低**：减速度低于预期但仍有一定减速效果
  - 核心特征：能减速但效果不足

- **在相反方向非预期的纵向移动**：车辆向与驾驶员请求方向相反的方向移动
  - 核心特征：请求前进却后退，或请求后退却前进
  - 适用场景：驱动系统方向控制错误（如变速箱挂错挡）
  - **不适用场景**：功能内部状态方向错误（如EPB拉起变释放）导致的车辆移动，应使用"非预期的纵向移动"

- **非预期的俯仰运动（滚动）**：车辆非预期的俯仰角变化

### 侧向运动详细定义
- **非预期的侧倾运动**：车辆侧倾角异常变化
- **非预期的横摆运动**：车辆非预期旋转或转向
- **失去横向运动**：无法实现侧向移动或转向
- **非预期的侧倾运动（滚动）**：车辆发生侧翻倾向

### 垂向运动详细定义
- **非预期的垂直运动**：车辆非预期的上下运动
- **非预期的横摆运动（滚动）**：垂向与横摆耦合的异常运动

## 3. 常见映射逻辑
### 驻车/保持/约束类功能
- 功能丧失：无法实现驻车/保持 → "非预期的纵向移动"（车辆会溜车，不是无法移动）
- 输出过小：夹紧力/保持力不足 → "非预期的纵向移动"
- 非预期激活：行驶中意外制动 → "非预期的减速"
- 方向错误：请求拉起却释放 → "非预期的纵向移动"（车辆可能溜车）

### 动力/驱动类功能
- 非预期输出：意外产生扭矩 → "非预期的加速"
- 输出过大：扭矩过大 → "非预期的加速"
- 输出过小：扭矩不足 → "非预期的失去加速" 或 "无法纵向移动"
- 功能丧失：无动力输出 → "无法纵向移动"
- 方向错误：请求前进却后退 → "在相反方向非预期的纵向移动"

### 制动类功能
- 非预期激活：意外制动 → "非预期的减速"
- 制动力过大 → "非预期的减速"
- 制动力不足 → "非预期车辆减速上的降低" 或 "非预期的失去减速"
- 功能丧失：无法制动 → "非预期的失去减速"

### 信息告警相关
- 驾驶员获得错误状态、错误报警：通常映射为"驾驶员获取错误的信息"。
- 应报警而未报警：通常映射为"失去报警"。
- 其他交通参与者因灯光、声光提示、外部显示等误判：通常映射为"其他驾驶员/行人误判"。
