# EPB Function Introduction

## 1. 系统概述

EPB（Electronic Parking Brake，电子驻车制动系统）用于在车辆静止、驻车、坡道停车、熄火停车、起步释放等场景下提供驻车制动力，防止车辆发生非预期纵向移动。

EPB 通常与 ESC、VCU、BCM、仪表、车身域控、制动执行器、轮速传感器、坡度估算、驾驶员开关输入等交互。

EPB 改变的是车辆纵向运动状态和驾驶员对车辆是否会保持静止的判断。它通常不应被仅理解为一个执行器，而应被理解为一个由请求、条件判断、执行、反馈和状态指示共同构成的车辆级功能。

## 2. 功能边界

EPB 主要关注车辆静止或低速起步/释放场景下的驻车制动控制，也可能在异常情况下影响行驶中纵向减速或车辆稳定性。

EPB 的基本安全属性包括：
- 需要驻车时能够建立足够驻车能力。
- 驻车建立后能够在预期条件内保持车辆静止。
- 满足释放条件时能够按驾驶员意图释放。
- 不满足释放条件时不应自行解除驻车。
- 系统状态应能让驾驶员正确理解车辆是否处于驻车保持状态。

## 3. 典型子功能

| 功能 ID | 子功能 | 功能说明 | 备注 |
|---|---|---|---|
| EPB_fc01 | 静态开关拉起 | 驾驶员在车辆静止或驻车条件满足时通过开关请求拉起 EPB。 |  |
| EPB_fc02 | 熄火自动拉起 | 车辆熄火或下电时自动拉起 EPB，防止车辆移动。 |  |
| EPB_fc03 | 起步自动释放 | 车辆起步条件满足时自动释放 EPB，避免拖滞并支持平顺起步。 |  |
| EPB_fc04 | 坡道驻车保持 | 在坡道驻车时保持足够夹紧力，防止车辆溜坡。 |  |
