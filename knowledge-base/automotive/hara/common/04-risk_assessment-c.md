# HARA 风险评估 - 可控性 C

## 使用方式

本文件是可控性 C 的权威知识库。只在判断 `控制能力 'C'`、`C-解释` 或 `C评级推理` 时读取。

C 评价相关人员避免伤害的现实能力，不只评价系统是否有告警。

## 可控性分类（C）

### C0：一般可控

定义：通常可由相关人员通过常规行为避免伤害，或危害事件不会导致不可接受的人身伤害风险。C0 通常不用于 ASIL 分配。

特征：

- 对驾驶员或相关人员非常明显
- 有充足反应时间
- 简单的纠正动作
- 多种感官提示
- 所需技能极小

示例：

- 单个雨刷停止工作（另一个仍工作）
- 大灯亮度降低（非完全失效）
- 燃油表不准（范围警告仍有效）
- 座椅加热器温度略微偏离目标
- 收音机音量突然变化

所需操作：简单补偿，明显检测

### C1：简单可控

定义：简单可控。大多数驾驶员或相关人员能够通过直接、常规的反应避免伤害。

特征：

- 容易注意到
- 有足够时间反应（> 5 秒）
- 直接的纠正动作
- 清晰的感官反馈
- 普通驾驶员技能即可

示例：

- ABS 降级（单个车轮）
- 动力转向助力减少（非完全丧失）
- 巡航控制需要多次尝试取消
- 自动变速箱延迟换挡（< 2 秒）
- 胎压监测误报警
- 电子驻车制动释放延迟

所需操作：简单反应，普通技能水平

### C2：通常可控

定义：通常可控。部分驾驶员或相关人员可以通过及时、适当的反应避免伤害，但需要更高注意力、空间或操作能力。

特征：

- 注意时可注意到
- 反应时间有限（1-5 秒）
- 需要熟练的驾驶员响应
- 可能需要训练有素的反应
- 需要熟悉的驾驶操作

示例：

- 动力转向完全丧失（行驶中）
- ABS 完全失效（仅能手动制动）
- ESC 降级性能（减少干预）
- 牵引力控制间歇性工作
- 发动机功率降低（非完全丧失）
- 自动变速箱卡在某一档
- 再生制动意外增加

所需操作：熟练驾驶，立即注意

### C3：难以控制或不可控

定义：难以控制或不可控。多数驾驶员或相关人员难以及时采取有效行动避免伤害。

特征：

- 可能不会立即注意到
- 反应时间极短（< 1 秒）
- 需要专业驾驶员技能
- 需要反直觉的响应
- 身体限制（力量/反应）
- 多个同时失效

示例：

- 制动完全失效（所有系统）
- 高速公路速度下转向完全锁死
- 高速公路速度下非预期全速加速
- 弯道中 ESC 非预期强力制动干预
- 行驶中安全气囊意外展开
- 四驱非预期扭矩分配导致打滑
- 高速公路速度下自动转向错误方向
- 悬架突然完全坍塌

所需操作：专业技能，可能不可控

## 判断锚点

判断时必须考虑感知来源、反应时间、可用操作、空间约束、驾驶员是否在车上以及道路使用者是否能避让。告警只能作为因素之一，不能自动等价为可控。

- C0/C1：风险容易被感知，反应窗口充足，普通驾驶员可用直接操作纠正。
- C2：需要及时识别并做出熟练反应，通常仍存在可行控制动作。
- C3：反应时间极短、空间受限、动作反直觉或驾驶员根本不在控制回路中。
